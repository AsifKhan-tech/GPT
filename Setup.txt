# Steps to create the Project

Chapter-0: Project initialization and shadcn setup

1. Initialize the Next.js Project
    pnpm create next-app@latest
    
2. Setup shadcn UI
    pnpm dlx shadcn@latest init --preset b27I2r68 --template next

    1 Install all components
        pnpm dlx shadcn@latest add


Chapter-1: Setup theme and Query provider

1. Visit shadcn & search for dark mode
    pnpm add next-themes
and follow the steps as mentioned in documentation 

- For adding toggle mode
    - Create a file inside components --> ui --> mode-toggle.tsx

2. Setup Tanstack Query(Help to manage server side states)
    pnpm add @tanstack/react-query

create a file components --> providers --> query-provider.tsx

Chapter-2: Prisma and Database Setup

   1- pnpm add prisma @prisma/client @prisma/adapter-pg
   2- pnpm prisma init :- To Initialize prisma in the project
   3- lib/db.ts :- for sample migration, Prisma Client initialization here
   4- pnpm prisma generate :- generate prisma client inside lib/generated/prisma...
   5- pnpm prisma migrate dev :- generate migrations folder inside prisma folder
 
prisma - The Prisma CLI for running commands like prisma init, prisma migrate, and prisma generate.
@prisma/client - The Prisma Client library for querying your database.
@prisma/adapter-pg - The node-postgres driver adapter that connects Prisma Client to your database.
